Super opinionated [ESLint](https://eslint.org/) configuration.

## Installation

```bash
npm install @comandeer/eslint-config --save-dev
```

## Usage

Put it inside your `.eslintrc.json` file:

```json
{
	"extends": "@comandeer/eslint-config"
}
```
